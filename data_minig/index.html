Data Mining Portfolio

by Trevor Whitney

This portfolio is a summary of the work I've done in the field of data mining. It is the result of a semester long course that I took as a graduate student at the Colorado School of Mines. All of the work from the semester is available in this github repository if you care to browse it. The first 6 items in this portfolio are academic explorations of various topics related to data mining. The last three sections are applications of this knowledge.
